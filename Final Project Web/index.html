Alex Enze Liu (alexliu0809@uchicago.edu)

Hi, this is Alex and Benyan. Our idea for the final project is to build Overwatch Helper, an application designated to provide userful and helpful information for overwatch players. The program will have a similar user interface as assignment 4. The application will mainly be divided into 4 sections: News, Heros Introductions, Map Introducations, and player statistics.
